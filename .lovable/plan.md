

## Plan : Suppression du Favicon

### Action
Supprimer le fichier `public/favicon.ico` du projet.

### Impact
- Le navigateur n'affichera plus d'icône personnalisée dans l'onglet
- Une icône par défaut du navigateur (généralement une page blanche ou icône générique) sera affichée à la place
- Aucun impact sur le fonctionnement de l'application

### Note
Si vous souhaitez remettre une icône plus tard, vous pourrez :
1. Ajouter un nouveau fichier favicon dans le dossier `public/`
2. Utiliser votre logo existant (`src/assets/admin-logo.png`) comme favicon
